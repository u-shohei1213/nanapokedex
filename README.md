# ななポケ図鑑

## 概要

## 機能
- ポケモンごとのイラスト一覧表示
- タイプ・地方・キーワード等による検索
- イラストの詳細表示
- ユーザー登録 / ログイン
- いいね・コメント機能

## 課題点
- SQLクエリの最適化<br>
現状のデータ量では応答速度に大きな問題はないものの、ループ内クエリを使用しているため、データ件数増加時のパフォーマンス低下が懸念される。
- 表示領域の動的制御<br>
表示データ量が増加した場合、描画コストや初回ロード時間への影響が想定されるため、スクロール位置に応じた描画制御などの対策が必要になる。
- ソースコードの責務分離<br>
一部のファイルに複数の機能の実装が集中しているため、保守性が低い。
