# ななポケ図鑑

## 概要
ポケモン図鑑とXのポストを連携したイラスト一覧アプリ

![一覧画面](https://github.com/user-attachments/assets/d4622358-30a4-4f1c-b6ea-738e858a7fd5)

![詳細画面](https://github.com/user-attachments/assets/182661cc-48de-4115-8864-0a8a52400a4b)

## 機能
- ポケモンごとのイラスト一覧表示
- タイプ・地方・キーワード等による検索
- イラストの詳細表示
- ユーザー登録 / ログイン
- いいね・コメント機能

## 技術構成
### フロントエンド
- React (Vite)
- TypeScript
- CSS
### バックエンド
- FastAPI (Python)
### データベース
- PostgreSQL
### インフラ
- AWS (Amplify / Elastic Beanstalk / RDS / API Gateway)
- Terraform
